# Attribution (3D assets)

This project intentionally does not ship third-party 3D assets by default.

If you add a model (e.g. from Sketchfab) under a CC license, record the attribution here.

Template:
- Asset name:
- Author:
- Source URL:
- License:
- Changes made:

Example:
- Asset name: Heart Anatomy
- Author: <name>
- Source URL: https://sketchfab.com/...
- License: CC BY 4.0
- Changes made: Converted to GLB, fixed scale, adjusted materials
